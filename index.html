Copernicana

Aplikacja do zarządzania grupami wycieczkowymi

Wykorzystane technologie: Java, JSP, Spring Boot 1.58, Spring Security, MySQL, H2, Bootstrap 3

Program dla touroperatora ułatwiający zarządzanie grupami turystów. Z punktu widzenia początkującego developera najciekawsze było użycie funkcji Spring Profiles. Projekt podzieliłem na profile dev, staging oraz production z odpowiednimi branchami git oraz stosownymi aplikacjami na Heroku.

Tabele w warstwie widoku generowane są za pomocą biblioteki Datatables, pozostałe komponenty za pomocą Boostrapa. Dodatkowo wybrane funkcjonalności realizowane są poprzez AJAX.

Soft creation

Strona www producenta poduszek dekoracyjnych

Zakres pracy obejmował stworzenie animacji loga w formacie svg oraz lokalizację zawartości.

Logo animowane jest za pomocą biblioteki vivus.js, informacja o wyświetleniu animacji przechowywana jest w local storage tak aby animacja nie pokazywała się po przeładowaniu strony. Lokalizacja teskstów wykonana jest za pomcą pluginu localize.js, który pozwala serwować treści w wybranym języku, przechowywane w formacie json. Automatyczny wybór języka na podstawie języka przeglądarki realizowany jest własnym skryptem, podobnie tłumaczenie placeholderów w formularzu kontaktowym.
