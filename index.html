Copernicana

Aplikacja do zarzdzania grupami wycieczkowymi

Wykorzystane technologie: Java, JSP, Spring Boot 1.58, Spring Security, MySQL, H2, Bootstrap 3

Program dla touroperatora uatwiajcy zarzdzanie grupami turystw. Z punktu widzenia pocztkujcego developera najciekawsze byo uycie funkcji Spring Profiles. Projekt podzieliem na profile dev, staging oraz production z odpowiednimi branchami git oraz stosownymi aplikacjami na Heroku.

Tabele w warstwie widoku generowane s za pomoc biblioteki Datatables, pozostae komponenty za pomoc Boostrapa. Dodatkowo wybrane funkcjonalnoci realizowane s poprzez AJAX.

Soft creation

Strona www producenta poduszek dekoracyjnych

Zakres pracy obejmowa stworzenie animacji loga w formacie svg oraz lokalizacj zawartoci.

Logo animowane jest za pomoc biblioteki vivus.js, informacja o wywietleniu animacji przechowywana jest w local storage tak aby animacja nie pokazywaa si po przeadowaniu strony. Lokalizacja teskstw wykonana jest za pomc pluginu localize.js, ktry pozwala serwowa treci w wybranym jzyku, przechowywane w formacie json. Automatyczny wybr jzyka na podstawie jzyka przegldarki realizowany jest wasnym skryptem, podobnie tumaczenie placeholderw w formularzu kontaktowym.
